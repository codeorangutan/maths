# Lesson 07 (Elite Track)

## Program Stage
- Year: 10 (accelerated stream)
- Term: 1
- Week: 3
- Session: Lesson 7
- Duration: 100 minutes

## Lesson Title
Trigonometric Ratios, Exact Values, and Modelling Foundations

## Lesson Purpose
Establish exact-value fluency and applied trigonometric modelling needed for advanced methods and specialist geometry.

## Learning Intentions
By the end of this lesson, the student will:
1. Use exact trig values and identities efficiently.
2. Solve right-triangle and non-right-triangle problems with correct method selection.
3. Interpret sine/cosine models in contextual settings.
4. Communicate angle and unit logic precisely.

## Success Criteria
- Core set: `>= 85%`
- Challenge set: `>= 70%`
- Timed section completed with full unit/angle notation
- Method choice justified in writing

---

## Minute-by-Minute Flow (100 min)

### 0-10 min: Retrieval Warm-Up
1. Exact \(\sin 30^\circ\)
2. Exact \(\cos 60^\circ\)
3. Exact \(\tan 45^\circ\)
4. Solve \(\sin\theta=\dfrac12\) for \(0^\circ\le\theta\le360^\circ\)
5. One sentence: when to use cosine rule

### 10-30 min: Explicit Teaching
- Exact values and ASTC logic.
- Sine rule and cosine rule selection.
- Area formula \(\dfrac12ab\sin C\).
- Intro to sinusoidal model parameters.

### 30-55 min: Guided Core Set
1. Evaluate exactly: \(2\sin30^\circ+\cos60^\circ\).
2. Solve: \(2\cos\theta=1\), \(0^\circ\le\theta<360^\circ\).
3. In triangle, \(a=9,b=12,C=40^\circ\), find \(c\).
4. Same triangle, find area.
5. A ladder 5 m long reaches wall at angle \(68^\circ\) to ground. Find height reached.
6. Find amplitude, period, and midline of \(y=3\sin(2x)-1\).

### 55-70 min: Singapore-Style Non-Routine Block
1. Prove \(\sin^2\theta+\cos^2\theta=1\) from right-triangle definitions.
2. Solve exactly: \(\sin x=\cos x\), \(0^\circ\le x<360^\circ\).
3. Given \(\sin\theta=\dfrac35\), \(\theta\) in QII, find \(\cos\theta,\tan\theta\).

### 70-85 min: Timed ATAR-Style Mini Test
1. Solve: \(\tan\theta=-\sqrt3\), \(0^\circ\le\theta<360^\circ\).
2. Triangle with sides 8, 11 and included angle \(47^\circ\): find third side.
3. Find area of same triangle.

### 85-100 min: Exit Ticket
1. Exact \(\sin150^\circ\)
2. Solve \(\cos\theta=0\), \(0^\circ\le\theta\le360^\circ\)
3. One sentence: why unit mode (degrees/radians) matters.

---

## Core Solutions
1. \(2\cdot\dfrac12+\dfrac12=\dfrac32\)
2. \(\cos\theta=\dfrac12\Rightarrow \theta=60^\circ,300^\circ\)
3. \(c^2=9^2+12^2-2(9)(12)\cos40^\circ\Rightarrow c\approx7.73\)
4. Area \(=\dfrac12(9)(12)\sin40^\circ\approx34.71\)
5. Height \(=5\sin68^\circ\approx4.64\,m\)
6. Amplitude 3, period \(\pi\), midline \(y=-1\)

## Timed Test Solutions
1. \(\theta=120^\circ,300^\circ\)
2. \(c\approx8.19\)
3. Area \(\approx32.18\)

## Exit Ticket Answers
1. \(\dfrac12\)
2. \(90^\circ,270^\circ\)
3. Wrong mode gives systematically wrong values.
