# Lesson 04 (Elite Track)

## Program Stage
- Year: 10 (accelerated stream)
- Term: 1
- Week: 2
- Session: Lesson 4
- Duration: 100 minutes

## Lesson Title
Exponential and Logarithmic Foundations for Methods Acceleration

## Lesson Purpose
Build fluent movement between exponential and logarithmic forms, with strong equation-solving accuracy and interpretation discipline.

## Learning Intentions
By the end of this lesson, the student will:
1. Apply index laws accurately in non-routine simplification.
2. Convert between exponential and logarithmic forms without prompts.
3. Solve basic exponential and logarithmic equations with valid restrictions.
4. Interpret growth/decay context with mathematically precise language.

## Success Criteria
- Core set: `>= 85%`
- Challenge set: `>= 70%`
- Timed mini test completed in target time
- All solutions include domain restrictions where relevant

---

## Minute-by-Minute Flow (100 min)

### 0-10 min: Retrieval Warm-Up
1. Simplify: \(2^{-3}\cdot 2^5\)
2. Solve: \(3^x=81\)
3. Rewrite \(\log_2 16\)
4. Evaluate: \(\dfrac{5^{-1}+5^{-2}}{5^{-3}}\)
5. State one index law and one log law

### 10-30 min: Explicit Teaching
- Index law architecture and common error traps.
- Conversion rule: \(a^b=c \iff \log_a(c)=b\).
- Log laws: product, quotient, power.
- Legality conditions: base \(>0\), base \(\neq 1\), argument \(>0\).

### 30-55 min: Guided Core Set
1. Simplify: \(\dfrac{3^{x+2}}{3^{x-1}}\)
2. Solve: \(5^{2x-1}=125\)
3. Solve: \(\log_3(x-1)=2\)
4. Solve: \(\log_2 x+\log_2(x-2)=3\)
5. Convert to log form: \(7^y=\dfrac{1}{49}\)
6. Convert to exponential form: \(\log_4\left(\dfrac{1}{64}\right)=k\)

### 55-70 min: Singapore-Style Non-Routine Block
1. If \(a^x=b\) and \(a^y=c\), express \(\log_b c\) in terms of \(x,y\).
2. Show \(\log_a b\cdot \log_b a=1\).
3. Solve: \(2^x+2^{x+1}=48\).

### 70-85 min: Timed ATAR-Style Mini Test
1. Solve: \(9^{x+1}=27^{2x-1}\)
2. Solve: \(\log_5(2x+1)=2\)
3. A population model is \(P(t)=1200(1.08)^t\). Find \(t\) when \(P(t)>2000\).

### 85-100 min: Exit Ticket
1. Solve: \(\log_2(x+3)=4\)
2. Simplify: \(\log_3 81-\log_3 3\)
3. One sentence: why must \(\log_4(x-7)\) satisfy \(x>7\)?

---

## Core Solutions
1. \(3^3=27\)
2. \(2x-1=3\Rightarrow x=2\)
3. \(x-1=9\Rightarrow x=10\)
4. \(\log_2[x(x-2)]=3\Rightarrow x(x-2)=8\Rightarrow x=4,-2\), valid \(x>2\), so \(x=4\)
5. \(y=-2\)
6. \(4^k=\dfrac{1}{64}=4^{-3}\Rightarrow k=-3\)

## Timed Test Solutions
1. \(3^{2x+2}=3^{6x-3}\Rightarrow 2x+2=6x-3\Rightarrow x=\dfrac{5}{4}\)
2. \(2x+1=25\Rightarrow x=12\)
3. \((1.08)^t>\dfrac{5}{3}\Rightarrow t>\dfrac{\ln(5/3)}{\ln(1.08)}\approx 6.62\), so smallest integer \(t=7\)

## Exit Ticket Answers
1. \(x+3=16\Rightarrow x=13\)
2. \(4-1=3\)
3. Log arguments must be strictly positive.

## Homework

### Core
1. Solve: \(2^{x+2}=64\)
2. Solve: \(\log_7(3x-2)=1\)
3. Solve: \(\log_2 x+\log_2(x-6)=4\)
4. Simplify: \(\dfrac{10^{n+1}}{10^{n-2}}\)
5. Convert \(\log_9\left(\dfrac{1}{27}\right)\) to exponential form and evaluate.

### Extension
6. Solve: \(3^{2x}-10\cdot3^x+9=0\)
7. Prove \(\log_a b=\dfrac{1}{\log_b a}\) from first principles.
