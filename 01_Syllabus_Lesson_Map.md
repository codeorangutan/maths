# Elite Maths Syllabus Lesson Map (Years 10-12)

## Structure
- Total program lessons: 72 (suggested baseline)
- Format: 3 lessons/week (`A` Concept, `B` Fluency, `C` Timed Problem Solving)
- File naming convention: `Lesson_XX.md`

## Year 10 (Acceleration Foundation) - Lessons 01-24

### Block 1: Baseline and Algebra Core (L01-L03)
- L01 Diagnostic + operating standards
- L02 Algebraic structure and identities
- L03 Quadratics, discriminant, optimization

### Block 2: Functions and Modelling Foundations (L04-L06)
- L04 Exponential/log foundations
- L05 Transformations, inverses, composition
- L06 Coordinate geometry and locus

### Block 3: Trig, Probability, Sequence Architecture (L07-L09)
- L07 Trigonometric exact values + modelling
- L08 Probability and conditional logic
- L09 Sequences, series, and proof patterns

### Block 4: Calculus Readiness and Mixed Consolidation (L10-L12)
- L10 Intro differentiation and rates
- L11 Intro integration and area
- L12 Mixed mastery + repair cycle

### Block 5: Advanced Year 10 Bridge (L13-L18)
- Polynomial division and remainder theorem
- Rational function behaviour and asymptotes
- Log/exponential modelling depth
- Intro vectors in 2D
- Advanced trigonometric equations
- Mid-year mixed exam + analysis

### Block 6: Pre-Methods Specialist Bridge (L19-L24)
- Binomial expansion foundations
- Parametric thinking introduction
- Further calculus readiness (chain/product intuition)
- Probability distributions intro
- Complex-number pre-foundations
- End-year mastery examination

## Year 11 (Methods + Specialist Core) - Lessons 25-48

### Block 7: Functions and Calculus Unit 1-2 (L25-L30)
- Advanced function transformations
- Domain/range and inverse depth
- Differential calculus techniques
- Applications of derivative
- Integral basics and areas
- Mixed calculus assessment

### Block 8: Trigonometry and Algebraic Depth (L31-L36)
- Trig identities and manipulation

- Exponential/log equations advanced
- Polynomial methods and modelling
- Proof and argumentation drills
- Mixed test with timed constraints

### Block 9: Specialist Foundations (L37-L42)
- Vectors (operations, geometry)
- Complex numbers (algebraic form)
- Matrices foundations
- Transformations and linear systems
- Kinematics-style modelling links
- Mixed Specialist checkpoint

### Block 10: Probability and Statistics Methods Depth (L43-L48)
- Discrete random variables
- Continuous distributions intro
- Normal distribution applications
- Statistical modelling and inference setup
- Exam-style interpretation tasks
- Year 11 mastery exam
