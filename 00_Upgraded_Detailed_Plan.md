# Elite Mathematics Program (Years 10-12)

## Program Goal
Build a mathematically exceptional student toward top-100 state rank performance in WA ATAR mathematics by combining:
- WA pathway alignment (`Methods` + `Specialist`)
- Singapore-style mastery pedagogy (deep structure, non-routine problems, spaced retrieval)
- Competition-level extension (proof, combinatorics, number theory, functional reasoning)

## Performance Standard
A lesson is considered mastered only when all are met:
- Core accuracy: `>= 85%`
- Challenge accuracy: `>= 70%`
- Timed section: on or under target time
- Written reasoning: logically complete, mathematically precise

## Program Design (Years 10-12)

### Year 10 (Acceleration Foundation)
Primary objective: build irreversible algebraic and functional fluency before full ATAR abstraction.

Core strands:
1. Algebraic structure and symbolic fluency
2. Functions and graph transformations
3. Polynomial, rational, exponential, and logarithmic pre-foundations
4. Trigonometric fluency and geometric reasoning
5. Probability/combinatorics introduction
6. Proof and argument construction

Expected outputs by end of Year 10:
- Can solve unfamiliar multi-step algebra without scaffolds
- Can justify solutions rigorously
- Can move between graphical, algebraic, and verbal representations quickly

### Year 11 (Methods + Specialist Unit 1-2 + selective Unit 3 bridge)
Primary objective: complete full Year 11 depth at high distinction pace and begin Year 12-level style.

Core strands:
1. Advanced functions and transformations
2. Differential and integral calculus fundamentals
3. Trigonometric identities and equations
4. Vectors, matrices, and complex numbers (Specialist foundations)
5. Discrete/continuous probability and statistical modelling

Expected outputs by end of Year 11:
- Distinction-level performance on combined Methods/Specialist papers
- Fast transfer between topics under timed pressure
- Strong error diagnosis and independent correction loops

### Year 12 (Top-100 Execution Year)
Primary objective: maximize exam performance quality and consistency.

Core strands:
1. High-complexity mixed papers (Methods + Specialist)
2. Advanced modelling and interpretation
3. Statistical inference, calculus applications, vector/complex integration
4. Exam-specific strategy (question triage, pacing, verification)

Expected outputs by end of Year 12:
- Stable high percentile scores across full mock sequences
- High confidence and low variance in exam performance

## Weekly Instruction Model
- `Lesson A` (Concept + Derivation): deep explicit instruction and proof
- `Lesson B` (Fluency + Technique): high-volume deliberate practice
- `Lesson C` (Problem Solving + Timed): non-routine and exam simulation

Weekly non-negotiables:
- 20-30 min cumulative retrieval
- 1 timed mini-assessment
- 1 written reasoning/proof task
- Error log update and targeted repair

## Assessment Architecture

### Baseline and Progression
- Initial diagnostic profile across 6 strands:
  - Number/Algebra
  - Functions/Graphs
  - Trigonometry/Geometry
  - Calculus readiness
  - Probability/Statistics
  - Reasoning/Proof
- Fortnightly micro-benchmarks
- End-of-term mastery exam (core + challenge + timed sections)

### Marking Bands
- `Band 1 (Elite)`: 90-100
- `Band 2 (Strong)`: 80-89
- `Band 3 (Developing)`: 70-79
- `Band 4 (At-risk for top-100 target)`: <70

## Lesson Blueprint (Template Used for Every Lesson)
1. Learning intention and success criteria
2. Prerequisite retrieval
3. Explicit teaching with worked examples
4. Guided core set
5. Non-routine Singapore-style challenge set
6. Timed ATAR-style mini test
7. Exit ticket with mastery threshold
8. Homework (core + extension + written reasoning)
9. Teacher diagnostic notes (error categories and intervention)

## Error Management Protocol
Every error is tagged as:
- `C` Conceptual
- `P` Process/strategy
- `A` Arithmetic/algebraic execution
- `R` Reading/interpretation
- `T` Time management

Repair loop:
1. Identify error type
2. Rewrite correct method
3. Reattempt similar item in 24 hours
4. Recheck in weekly retrieval

## Top-100 Strategy Layer
- Prioritize difficult mixed-topic questions early in training, not only near exams
- Train proof-quality communication from Year 10
- Use high-frequency timed drills to reduce cognitive load under pressure
- Maintain balanced profile: avoid over-specialization in favorite strands
- Build consistency, not one-off peak scores

## First Block Scope
The first 3 lessons (provided separately) establish:
- rigorous baseline profile
- algebraic structure fluency
- elite problem-solving and reasoning habits

These lessons are designed to be the reset point for the full program.
