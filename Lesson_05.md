# Lesson 05 (Elite Track)

## Program Stage
- Year: 10 (accelerated stream)
- Term: 1
- Week: 2
- Session: Lesson 5
- Duration: 100 minutes

## Lesson Title
Function Transformations, Inverses, and Composite Structure

## Lesson Purpose
Develop fast, accurate function manipulation for Methods/Specialist readiness: transformations, composition, inversion, and domain-range control.

## Learning Intentions
By the end of this lesson, the student will:
1. Describe and apply graph transformations precisely.
2. Compute and simplify composite functions.
3. Determine inverse functions with domain conditions.
4. Translate between symbolic, graphical, and contextual function descriptions.

## Success Criteria
- Core set: `>= 85%`
- Challenge set: `>= 70%`
- Timed section complete with all notation correct
- Domain/range conditions explicitly stated for inverse tasks

---

## Minute-by-Minute Flow (100 min)

### 0-10 min: Retrieval Warm-Up
1. Evaluate: \(f(3)\) for \(f(x)=2x^2-1\)
2. Solve: \(g(g(x))\) for \(g(x)=x+4\)
3. State condition for invertibility on an interval
4. Give domain of \(\dfrac{1}{x-2}\)
5. Find \(f^{-1}(7)\) if \(f(x)=2x+1\)

### 10-30 min: Explicit Teaching
- Transformation grammar: \(y=f(x-a)+b\), \(y=af(x)\), \(y=f(bx)\), \(y=-f(x)\).
- Composite workflow and simplification structure.
- Inverse algorithm: swap \(x,y\), solve, then domain check.
- Restricting domain to make quadratics invertible.

### 30-55 min: Guided Core Set
1. Describe transformations from \(y=x^2\) to \(y=-(x-3)^2+5\).
2. If \(f(x)=3x-2\), \(g(x)=x^2+1\), find \(f\circ g\) and \(g\circ f\).
3. Find \(f^{-1}(x)\) for \(f(x)=\dfrac{x-4}{3}\).
4. For \(h(x)=x^2-6x+10\), find \(h^{-1}(x)\) when \(x\ge 3\).
5. Determine domain of \((f\circ g)(x)\) if \(f(x)=\sqrt{x-1}\), \(g(x)=2x+5\).
6. Solve \((f\circ g)(x)=13\) for \(f(x)=x+1, g(x)=3x-2\).

### 55-70 min: Singapore-Style Non-Routine Block
1. Construct two different pairs \((f,g)\) with same composite \(f\circ g=2x+5\).
2. If \(f\circ f(x)=4x+3\) and \(f\) is linear, find \(f(x)\).
3. Show why \(x^2\) has no inverse on all reals but does on \([0,\infty)\).

### 70-85 min: Timed ATAR-Style Mini Test
1. Given \(f(x)=\dfrac{2x+1}{x-3}\), find \(f^{-1}(x)\).
2. If \(f(x)=x^2+2x\), find \(f(x+1)-f(x)\).
3. Determine domain of \(\sqrt{5-2x}\).

### 85-100 min: Exit Ticket
1. Find \((f\circ g)(2)\) for \(f(x)=x-3, g(x)=x^2\).
2. Find inverse of \(f(x)=5x-7\).
3. One sentence: why domain restriction matters for inverse functions.

---

## Core Solutions
1. Reflect in x-axis, shift right 3, up 5.
2. \(f\circ g(x)=3x^2+1\), \(g\circ f(x)=(3x-2)^2+1\)
3. \(f^{-1}(x)=3x+4\)
4. \(h(x)=(x-3)^2+1\Rightarrow h^{-1}(x)=3+\sqrt{x-1}\)
5. Need \(2x+5\ge1\Rightarrow x\ge -2\)
6. \((f\circ g)(x)=3x-1=13\Rightarrow x=\dfrac{14}{3}\)

## Timed Test Solutions
1. \(y=\dfrac{2x+1}{x-3}\Rightarrow yx-3y=2x+1\Rightarrow x(y-2)=3y+1\Rightarrow f^{-1}(x)=\dfrac{3x+1}{x-2}\)
2. \((x+1)^2+2(x+1)-(x^2+2x)=2x+3\)
3. \(5-2x\ge0\Rightarrow x\le \dfrac{5}{2}\)

## Exit Ticket Answers
1. \(f(g(2))=f(4)=1\)
2. \(f^{-1}(x)=\dfrac{x+7}{5}\)
3. Without one-to-one behavior, inverse is not a function.

## Homework

### Core
1. Describe transformation from \(y=\sqrt{x}\) to \(y=2\sqrt{x-4}-1\).
2. Compute \(f\circ g\) for \(f(x)=\dfrac{1}{x}\), \(g(x)=x-2\).
3. Find inverse of \(f(x)=\dfrac{x+5}{2}\).
4. Restrict domain of \(y=x^2-4x\) so inverse exists.
5. Solve \((f\circ g)(x)=0\) for \(f(x)=x+3\), \(g(x)=2x-5\).

### Extension
6. If \(f(x)=\dfrac{ax+b}{cx+d}\), investigate conditions for existence of inverse.
7. Prove composition of two one-to-one functions is one-to-one.
